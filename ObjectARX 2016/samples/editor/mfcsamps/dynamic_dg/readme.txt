This sample demonstrates how to use an MFC DLL in an ObjectARX application.

To use dynamic.arx:
1. Start AutoCAD.
2. Type the ARX command and load dynamic.arx.
4. Type the dynamicmfc command, defined by dynamic.arx.
5. A simple modal dialog box will be shown.

Refer to the ObjectARX Developer's Guide for more information.
